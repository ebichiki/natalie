好きな芸人のニュースを収集する
====

## Overview
お笑いナタリーのRSSから自分が好きな芸人の記事を表示する

## Usage
artistのリストに好きな芸人、コンビ名などを登録しておく。

## Install
feedparserモジュールをpipを使ってインストールする。

`pip install feedparser`
